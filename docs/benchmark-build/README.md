# Benchmark Build Notes

This folder is the working notebook for the benchmark runner and suite build-out.
It is separate from `runs/`, which contains operational outputs, logs, and local
artifacts that may be ignored or cleaned up.

Use this folder for:

- Runner reliability decisions.
- Live benchmark incident notes.
- Hardware/run-planning notes for RTX 5090 and RTX 6000 Pro.
- Backlog items required before asking third parties to run the benchmark.
- Handoffs for website or scorer updates that depend on benchmark behavior.
