local-bench
===========

This project includes frozen benchmark item sets derived from third-party
datasets, and adapts third-party verifier code. Each upstream carries its own
license, which continues to apply in full. The local-bench project's OWN source
license is not yet finalized (see the top-level LICENSE file).
